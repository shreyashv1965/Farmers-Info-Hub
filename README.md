# 🌾 Farmers Info Hub

Farmers Info Hub is a simple website built using **HTML, CSS, and JavaScript**.  
It provides information on modern farming techniques, tools, and challenges.  
The site includes an image slideshow, search bar, and easy navigation — designed to help users explore sustainable and technology‑based farming.

---

## 📖 Features
- **Image Slideshow**: Highlights farming practices and tools.
- **Search Bar**: Quickly find relevant farming information.
- **Responsive Design**: Easy navigation across devices.
- **Educational Content**: Focused on sustainable and modern farming methods.
